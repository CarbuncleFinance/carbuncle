### レイヤー構成

| レイヤー | 目的 | 対応チェーン |
| --- | --- | --- |
| 即時決済 | 高速・低手数料で店頭支払い | XRP Ledger / Xahau Network |
| 資産運用 | 預入・ステーキングで利回り獲得 | XRP EVM |
| ブリッジ | ２つのレイヤー間で資産移動 | Axeler |
